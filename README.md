<summary><b>🥽 Agriverse — Smart Farming Experience with Digital Twinning</b></summary>

Built during the Metaverse Internship at Munjal BCU Centre of Innovation & Entrepreneurship (Feb–May 2025). An immersive VR smart-farming simulation featuring teleportation systems, an agriculture museum, and a live digital twin synced with real-world sensors (ESP32/ESP8266, DHT11, LDR) for temperature, humidity, and light data via Firebase.

| Stack | Scale | Impact |
|---|---|---|
| Unity3D, Blender, C#, Firebase, ESP32/ESP8266, Arduino | Full VR environment + real-time IoT digital twin sync | Delivered as core internship output; hands-on XR + IoT integration |

<summary><b>📦 AI-Assisted Cartonization & Packing Optimization System</b></summary>
<br/>

An intelligent cartonization system for e-commerce logistics, tackling the NP-hard 3D Bin Packing problem via First Fit Decreasing, Best Fit, and 3D Bin Packing heuristics, enhanced with TensorFlow-based pattern analysis. Secured with Spring Security and role-based access, with an Angular frontend for real-time 3D visualization of packing.

| Stack | Scale | Impact |
|---|---|---|
| Java, Spring Boot, Spring Security, Angular, Python, TensorFlow | Heuristic + ML-assisted optimization engine | Near-optimal real-time packing suitable for warehouse/logistics use |

</details>
